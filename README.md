📊 Electronics Sales Analysis – Data Analyst Project

📌 Overview

This project explores electronics sales data to identify revenue trends, product performance, and customer purchasing behavior. Using Python (Jupyter Notebook) for analysis, the goal is to uncover actionable insights that can help optimize sales strategies and inventory management.


📂 Repository Structure

Electronic_sales.csv → Raw dataset containing product sales records (product name, category, quantity, price, revenue, etc.).

Electronics_Sales_Analysis.ipynb → Jupyter Notebook with step‑by‑step data cleaning, exploration, and visualization.


🛠️ Tools & Technologies

Python (Pandas, NumPy) → For data cleaning and manipulation.

Matplotlib & Seaborn → For data visualization.

Jupyter Notebook → For interactive analysis and reporting.


📊 Key Insights

Revenue Trends → Identifies peak sales periods and seasonal demand.

Top Products → Highlights best‑selling electronics contributing most to revenue.

Category Analysis → Compares performance across product categories (e.g., mobiles, laptops, accessories).

Customer Behavior → Examines purchase patterns and average spending.


📈 Sample Analysis

The notebook includes:

Data Cleaning → Handling missing values and formatting columns.

Exploratory Data Analysis (EDA) → Summary statistics and visualizations.

Revenue Breakdown → Total revenue by product and category.

Trend Charts → Monthly/quarterly sales performance.
